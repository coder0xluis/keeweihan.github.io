pip3正确升级姿势

最近用pip安装包的时候总是提示有新版本，太碍眼，不能忍

我安装的python3环境，对应pip3包管理工具，所以直接撸

$ sudo pip3 install --upgrade pip3

居然报错了，提示没有匹配的pip3版本，貌似姿势不正确

原来正确姿势是：

$ sudo pip3 install --upgrade pip

bingo ～！ 搞定，成功升级

用Scrapy做个简单的爬虫

Scrapy文档

官网
https://scrapy.org/

英文文档 （版本较新）
https://doc.scrapy.org/en/latest/

中文文档 （版本比英文版滞后）
https://scrapy-chs.readthedocs.io/zh_CN/0.24/index.html

我的环境：

  • python 3.6
  • scrapy 1.5.1

安装Scrapy

$ pip3 install Scrapy

创建Scrapy项目

$ scrapy startproject novel

将会生成如下项目结构目录

novel/
    scrapy.cfg
    novel/
        __init__.py
        items.py
        pipelines.py
        settings.py
        spiders/
            __init__.py
            ...
