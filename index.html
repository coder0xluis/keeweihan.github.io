Scrapy文档

官网
https://scrapy.org/

https://doc.scrapy.org/en/latest/

中文文档 （版本比英文版滞后）

我的环境：

  • python 3.6
  • scrapy 1.5.1

安装Scrapy

pip3 install Scrapy

创建Scrapy项目

scrapy startproject novel

将会生成如下项目结构目录

novel/
    scrapy.cfg
    novel/
        __init__.py
        items.py
        pipelines.py
        settings.py
        spiders/
            __init__.py
            ...
